# Escape Room (Text-Based Java Game)

## Overview

This is a personal project where I am building a very simple **text-based escape room**. To do that, I've chosen to use Java and Maven.

The goal of this project is to simulate a very simple escape room where the player needs to go through rooms, solve puzzles, collect items, and find a way to escape. All these actions are done through terminal commands.

---

## Game Concept

The game is fully text-based since the main point of this project was to apply the concepts I learned from OOP in a game. I used the escape room to apply polymorphism, encapsulation, low coupling, high cohesion, etc.

---

## Features

- 3 connected rooms
- Player inventory system
- Doors (locked and unlocked)
- Items needed to continue the game
- Combinations the player needs to find
- Game timer

---

## Summary of the game

The game has 3 rooms: Past, Present, and Future. In order to save the world and get out of there, you need to fix the 3 machines. Each machine gives a different Time Stone, and you need all 3 to leave and finalize the game.

In each room, you can find multiple objects, and through them, you can find the combination to activate the machine and restore the past, present, or future.

Once you've activated the machine, the game lets you go to the following room.

In order to win, you also need to make sure the timer doesn't run out since you only have 10 minutes to get out!

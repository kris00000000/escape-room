# Escape Room (Text-Based Java Game)

## Overview

This is a personal project where I am building a **text-based escape room game** using Java and Maven.

The goal of the project is to simulate a classic escape room experience where the player must explore rooms, solve puzzles, collect items, and find a way to escape. It will all be done through the terminal commands.

---

## Game Concept

The game is fully **text-based** and runs in the terminal.  

Each action affects the game world and helps the player progress through the escape room.

---

## Features (Planned / In Progress)

- Multiple connected rooms
- Player inventory system
- Doors (locked and unlocked)
- Keys and usable items
- Puzzle mechanics
- Game timer (time-based challenge)
- Command parser system
- Win/lose conditions

---

## Project Structure

The project will follow a simple object-oriented / MVC-inspired design:

- **controller** → Game logic and input handling.
- **model** → Game world.
- **view** → Output to the console. 
- **util/* → Helper classes. 

---

This project is mainly for learning and practice, focusing on:

- Object-Oriented Programming (OOP)
- Clean code structure
- Project architecture design
- Game logic development
